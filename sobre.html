Conheça a Linguagem FireScript

Sobre a Linguagem

Esta é uma linguagem simples, baseada na linguagem de programação C. Abaixo foi colocada uma simples descrição de como iniciar a programação em FireScript.

Comandos

O programa deve ser iniciado com { e finalizado com }.

Declaração de variáveis - Ocorre da seguinte forma nome_variavel tipo. O nome da variável deve ter apenas 1 caractere, sendo letra. Os tipos podem ser int, string ou boolean.

Atribuições - Funcionam assim: nome_varivel = valor, sendo que valor podem ser expressões matémáticas, como no seguinte exemplo: a = 1+2+3+4+5.

Para se exibir em tela o valor de uma variável é usado o comando show(nome_variavel).

If - O comando if é usado da seguinte forma: if(condicao_um == condicao_dois). após a declaração da condição, é necessário abrir chaves, e no fim fechar chaves { ... }. Dentro das chaves, vai o que deve acontecer caso a condição seja verdadeira. Como no exemplo abaixo:
if( a == b ) {
   int c
   c = 2 + a
   show(c)
}

While - O laço de repetição while é representado da seguinte forma: while(condicao_um == condicao_dois)após a declaração da condição, é necessário abrir chaves, e no fim fechar chaves { ... }. Dentro das chaves, deve ser colocado o que deve acontecer enquanto a condição for verdadeira. Como no exemplo abaixo:
while( a == b ) {
   string c
   c = "compilador"
   show(c)
}
